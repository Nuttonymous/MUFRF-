# 📋 คู่มือ Setup Google Sheet สำหรับหน้ากิจกรรม MUFRF

คู่มือนี้จะแนะนำวิธีสร้าง Google Sheet เพื่อใช้เป็นฐานข้อมูลกิจกรรมของเว็บไซต์ MUFRF
เมื่อ setup เสร็จแล้ว คุณสามารถเพิ่มกิจกรรมใหม่ลงใน Google Sheet ได้เลย
หน้าเว็บจะอัปเดตข้อมูลเองอัตโนมัติ (ใช้เวลาประมาณ 5 นาทีหลังแก้ไข)

---

## ขั้นตอนที่ 1: สร้าง Google Sheet ใหม่

1. เปิด [Google Sheets](https://sheets.google.com)
2. คลิก **+ Blank** เพื่อสร้าง Spreadsheet ใหม่
3. ตั้งชื่อไฟล์ เช่น `MUFRF Activities Database`

---

## ขั้นตอนที่ 2: ใส่ Header (แถวแรก)

ใส่ค่าต่อไปนี้ในแถวแรก (Row 1) ให้ตรงตามนี้ทุกประการ:

| คอลัมน์ | A    | B    | C     | D    | E    | F     |
|---------|------|------|-------|------|------|-------|
| **ค่า** | id   | date | title | desc | link | image |

> ⚠️ **สำคัญ**: ชื่อ header ต้องเป็นตัวพิมพ์เล็กทั้งหมด (`id`, `date`, `title`, `desc`, `link`, `image`) 
> ห้ามมีช่องว่างหน้า/หลัง และห้ามเปลี่ยนลำดับ

### คำอธิบายแต่ละคอลัมน์:

| คอลัมน์ | ความหมาย | ตัวอย่าง | จำเป็น? |
|---------|----------|---------|---------|
| **id** | รหัสกิจกรรม (ไม่ซ้ำกัน) | `rmutr30062026` | ✅ แนะนำ |
| **date** | วันที่จัดกิจกรรม | `วันที่ 30 มิถุนายน 2569` | ✅ แนะนำ |
| **title** | ชื่อกิจกรรม | `สถาบันวิจัยฯ เข้าเยี่ยมชม...` | ✅ **จำเป็น** |
| **desc** | รายละเอียดกิจกรรม | `ศูนย์เครื่องมือฯ ได้ต้อนรับ...` | ✅ แนะนำ |
| **link** | ลิงก์อ่านเพิ่มเติม | `https://example.com/news/1` | ❌ ไม่จำเป็น |
| **image** | URL รูปภาพ | `https://drive.google.com/file/d/xxx/view` | ❌ ไม่จำเป็น |

---

## ขั้นตอนที่ 3: ใส่ข้อมูลกิจกรรม

ใส่ข้อมูลกิจกรรมตั้งแต่แถวที่ 2 เป็นต้นไป ตัวอย่าง:

| id | date | title | desc | link | image |
|----|------|-------|------|------|-------|
| rmutr30062026 | วันที่ 30 มิถุนายน 2569 | สถาบันวิจัยฯ ราชมงคลสุวรรณภูมิ เข้าเยี่ยมชม | ศูนย์เครื่องมือฯ ได้ต้อนรับผู้เข้าเยี่ยมชม... | https://... | https://drive.google.com/file/d/abc123/view |
| sem-training | 15 กรกฎาคม 2569 | อบรม SEM สำหรับนักวิจัย | การฝึกอบรมการใช้งานกล้อง SEM... | | |

> 💡 **เคล็ดลับ**: ถ้าไม่มีลิงก์หรือรูปภาพ ปล่อยช่องนั้นว่างไว้ได้เลย ระบบจะจัดการให้อัตโนมัติ

---

## ขั้นตอนที่ 4: Publish to the Web

นี่คือขั้นตอนสำคัญที่สุด! ทำให้ Google Sheet กลายเป็น public CSV ที่เว็บไซต์ดึงข้อมูลได้

1. ไปที่เมนู **File** → **Share** → **Publish to web**
2. ในหน้าต่างที่ปรากฏ:
   - **Link**: เลือก `Entire document`
   - **Format**: เลือก **Comma-separated values (.csv)**
3. คลิกปุ่ม **Publish**
4. จะมี URL ปรากฏขึ้น หน้าตาประมาณนี้:
   ```
   https://docs.google.com/spreadsheets/d/e/2PACX-1vXXXXXXXXXXXXXXXXXXXXXXX/pub?output=csv
   ```
5. **คัดลอก URL นี้ไว้** (จะใช้ในขั้นตอนถัดไป)

> ⚠️ **สำคัญ**: ต้องเลือก format เป็น **CSV** เท่านั้น! ถ้าเลือก Web page จะใช้ไม่ได้

> 💡 หลังจาก Publish แล้ว ทุกครั้งที่แก้ไขข้อมูลใน Sheet ระบบจะอัปเดต CSV ให้อัตโนมัติ
> (อาจมี cache ประมาณ 5 นาที)

---

## ขั้นตอนที่ 5: ใส่ URL ลงในเว็บไซต์

1. เปิดไฟล์ `activities.html`
2. ค้นหาบรรทัดนี้ (อยู่ในส่วน `<script>` ด้านล่าง):
   ```javascript
   const GOOGLE_SHEET_CSV_URL = 'YOUR_GOOGLE_SHEET_CSV_URL_HERE';
   ```
3. แก้ไขเป็น URL ที่คัดลอกมาจากขั้นตอนที่ 4:
   ```javascript
   const GOOGLE_SHEET_CSV_URL = 'https://docs.google.com/spreadsheets/d/e/2PACX-1vXXXXXXXXX/pub?output=csv';
   ```
4. บันทึกไฟล์ เปิดหน้าเว็บ — เสร็จ! 🎉

---

## การใส่รูปภาพ

### วิธีที่ 1: ใช้รูปจาก Google Drive (แนะนำ)

1. อัปโหลดรูปภาพลง Google Drive
2. คลิกขวาที่ไฟล์ → **Share** → **General access** → เปลี่ยนเป็น **Anyone with the link**
3. คัดลอก link ที่ได้ (เช่น `https://drive.google.com/file/d/1AbCdEfGhIjKlMnOp/view?usp=sharing`)
4. วาง link นี้ลงในคอลัมน์ **image** ของ Google Sheet

> ✅ ระบบจะแปลง Google Drive URL ให้เป็น direct image URL อัตโนมัติ ไม่ต้องแปลงเอง!

### วิธีที่ 2: ใช้ URL สาธารณะอื่นๆ

วาง URL รูปภาพสาธารณะได้โดยตรง เช่น:
- `https://i.imgur.com/xxxxx.jpg`
- `https://example.com/images/photo.png`
- URL อื่นๆ ที่เข้าถึงได้โดยไม่ต้อง login

### วิธีที่ 3: ใช้รูปจากเซิร์ฟเวอร์เดิม (Local path)

ถ้าเว็บอยู่บนเซิร์ฟเวอร์เดียวกัน สามารถใส่ relative path ได้:
```
images/activities/my-photo.jpg
```

---

## วิธีเพิ่มกิจกรรมใหม่

หลังจาก setup เสร็จแล้ว เมื่อมีกิจกรรมใหม่ ทำเพียง:

1. เปิด Google Sheet
2. **เพิ่มแถวใหม่** ที่ด้านล่าง (หรือแทรกตรงไหนก็ได้)
3. กรอกข้อมูลกิจกรรม
4. **ไม่ต้องแก้ไขโค้ดใดๆ** — รอประมาณ 5 นาที หน้าเว็บจะอัปเดตเอง!

> 💡 **เคล็ดลับ**: ถ้าอยากให้กิจกรรมล่าสุดแสดงก่อน ให้ใส่กิจกรรมใหม่ไว้แถวบนสุด (แถวที่ 2 ต่อจาก header)

## แก้ไขปัญหาที่พบบ่อย

### 🔴 หน้าเว็บแสดง "ข้อมูลสำรอง (Offline)"

**สาเหตุ**: ยังไม่ได้ใส่ Google Sheet URL หรือ URL ผิด
**วิธีแก้**: ตรวจสอบว่า:
1. ได้ Publish to web เป็น CSV แล้ว
2. URL ลงท้ายด้วย `?output=csv`
3. วาง URL ถูกที่ใน `activities.html`

### 🔴 รูปภาพไม่ขึ้น

**สาเหตุ**: ไฟล์รูปไม่ได้เปิดเป็น public
**วิธีแก้**: ตรวจสอบการแชร์ไฟล์ใน Google Drive → ต้องเป็น "Anyone with the link"

### 🔴 ข้อมูลไม่อัปเดต

**สาเหตุ**: Google Sheets มี cache ประมาณ 5 นาที
**วิธีแก้**: รอสักครู่แล้วรีเฟรชหน้าเว็บ หรือกด Ctrl+Shift+R (hard refresh)

### 🔴 ข้อมูลในช่องรายละเอียดมีเครื่องหมายจุลภาค (comma)

**สาเหตุ**: ปกติ — Google Sheets จัดการ comma ใน CSV ให้อัตโนมัติ
**วิธีแก้**: ไม่ต้องทำอะไร ระบบ CSV parser จัดการให้แล้ว
